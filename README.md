# MDM Accelerator

Enterprise-grade Master Data Management (MDM) accelerator.

## Objective
To provide a configurable, reusable MDM framework
that can onboard multiple source systems and
produce golden master records.

## Architecture
- Source → Bronze → Silver
- Config-driven ingestion
- Rule-based and similarity-based matching

## Technology Stack
- Python
- PySpark
- Delta Lake

## Status
Local development (Databricks-compatible)
